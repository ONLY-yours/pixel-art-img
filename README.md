# Pixel Art Tools

这是一个个人向的仓库，主要是提供一些图像化像素风能力，使用 TypeScript 直接编写

个人还会添加更多能力：
- 各种像素化能力
- CRT 模拟
- 90S 做旧
- WIP

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
